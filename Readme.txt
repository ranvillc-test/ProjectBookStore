Proyecto E2E Book Store 12/08/2023

Proyecto de pruebas automatizado E2E sobre flujo de eliminación de un libro en el módulo de colecciones.
El proyecto Book Store consta de la ejecución de un escenario de prueba y dos casos de pruebas que complementan la funcionalidad
El proyecto se desarrolla bajo java versión "17.0.1"

Tener en cuenta para la ejecución del proyecto:
- Importar este proyecto como Gradle bajo lenguaje de programación Java
- La configuración de la ejecución se realiza con los comandos de RUN sobre el Runner ubicado en la ruta ..co.com.book.store.runners.Runner del proyecto

Para el informe de pruebas se puede abrir por consola una vez ejecutados los casos abriendo el archivo Main report: file:///C:/Users/ranvi/IdeaProjects/book-store/target/site/serenity/index.html al final de la ejecución del proyecto o también por medio de la carpeta ubicada en la ruta ..../target/site/serenity/index.html
